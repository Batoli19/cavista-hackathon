# Cavista Hackathon - Jarvis Project Assistant (Desktop)

## Modules
- ui/ : Desktop UI (CustomTkinter)
- voice/ : Speech-to-text + Text-to-speech
- engine/ : Project planning + tracking logic
- actions/ : Windows automation (minimize, open apps, open URLs)
- documents/ : Export to Word/Excel
- integrations/ : Optional git tools

## Quick start (CLI mode)
1) Install Python 3.11+
2) pip install -r requirements.txt
3) python main.py

## Example commands
- create project Hackathon Jarvis
- generate plan
- status
- delay t3 1
- done t2
- export plan to word
- export schedule to excel
- minimize all windows
- open notes
- open word
